Запустить новый сервис

Сервис — это запущенный контейнер, доступный по внутреннему порту. Позже его можно будет опубликовать, создав Приложение.

buildРасширенная сборка

Для не-Python проектов или своего порта. Пусто → Python-автоген (uvicorn main:app) на порту 80. Свой Dockerfile из архива имеет приоритет. Порт 0 = воркер/бот без сетевого порта. БД (postgres/redis) — лучше свой Dockerfile.

Опубликовать приложение

Приложение - это публичная точка входа (домен), которая указывает на один из ваших запущенных сервисов.
